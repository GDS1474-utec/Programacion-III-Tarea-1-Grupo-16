# Programacion-III-Tarea-1-Grupo-16
1. descargar el código: En la parte de Code > Download ZIP.
2. Extraer el ZIP: En tu explorador de archivo, busca el ZIP, le das a click derecho y extraer todo. Te debería soltar una carpeta.
3. Abres la carpeta en tu interpretador: Seleccionas la carpeta que has extraído. En Clion, File > Open file or project.
4. Crear el ejecutable (CMake) automáticamente: Arriba del código te saldrá "This file does not belong to any project target". Debes presionar al costado del mensaje: fix > Configure CMake Project > Create CMakeLists.txt. Al crear, revisa que estén marcados main.cpp, Tensor.cpp, Tensor.h, TensorTransform.cpp y TensorTransform.h.
5. Ejecutas el proyecto: Finalmente ejecutas el proyecto con el botón ▶️.
